 Documentación Técnica 

 Árbol General
Este módulo implementa un "árbol de directorios" similar a un sistema de archivos,
usando estructuras dinámicas en C++ moderno

El sistema permite crear, eliminar, mover y recorrer archivos y carpetas,
así como exportar la estructura a formato JSON.

  Nodo
Contiene:
- id
- nombre
- tipo
- contenido
- hijos
- padre

  Árbol
Contiene:
- root
- métodos CRUD
- JSON
- papelera
- búsqueda hash/map
- recorrido preorden

  Trie
Estructura para autocompletar nombres.
